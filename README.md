# Structure

extension/ contains results from the CO2 emissions sensitivty analysis
figures/ contains result images
results/ contains primary scenario analysis results
src/ contains core python model code

Jupyter Notebooks in root folder:
co2_sensitivity_extension.ipynb:
Used to run the co2 sensitivity analysis extension

data_cleaning.ipnyb:
Notebook for importing, cleaning, and re-exporting data

sandbox.ipynd:
Sandbox notebook - most recenty used to generate results figures
